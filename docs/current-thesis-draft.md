# Análisis de Alineación: Tesis - Proyecto HybridSecScan

## ALINEACIÓN PERFECTA IDENTIFICADA

### Objetivo General de Tesis:
"Desarrollar un sistema de auditoría automatizada para la detección de vulnerabilidades OWASP API Top 10 en APIs REST, que combine análisis estático (SAST) de especificaciones OpenAPI y configuraciones de API Gateways, con análisis dinámico (DAST) de endpoints activos, y que genere reportes ejecutivos en formatos PDF/JSON con métricas precisas de seguridad."

### Lo que HybridSecScan Implementa:
- Sistema de auditoría automatizada
- Detección OWASP API Top 10  
- Análisis híbrido (SAST + DAST)
- Análisis de especificaciones OpenAPI
- Análisis dinámico de endpoints
- Reportes automatizados
- Métricas de seguridad

## FORTALEZAS DE LA TESIS ACTUAL

### Marco Teórico Excepcional:
- 15 artículos relevantes analizados
- Cobertura completa de herramientas existentes
- Identificación clara de gaps en la literatura
- Excelente análisis de SAST vs DAST

### Justificación Sólida:
- Estadísticas actuales (Salt Security 2023: 78% organizaciones atacadas)
- Contexto peruano específico (MTC 2022)
- Alineación con ODS 9 y 16
- Necesidad empresarial clara

### Objetivos Específicos Bien Definidos:
1. Análisis de herramientas existentes - Ya cubierto en estado del arte
2. Evaluación de limitaciones - Implementado en sistema de comparación
3. Diseño de modelo híbrido - Arquitectura HybridSecScan
4. Implementación de prototipo - Sistema funcional
5. Validación experimental - Framework de evaluación
6. Comparación con herramientas - Sistema de benchmarking
7. Líneas futuras - Extensibilidad del sistema

## ELEMENTOS QUE POTENCIAN LA TESIS

- **Algoritmo de correlación inteligente** (contribución científica)
